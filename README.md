# AI 学习网站 - 四级进阶体系

一个系统化的 AI 学习网站，从小学到大学四级进阶课程体系。

## 🌐 在线访问

部署后访问：https://your-project.vercel.app

## 📚 课程结构

- **小学级** - AI 启蒙阶段 (3-6 个月)
- **初中级** - AI 入门阶段 (6-12 个月)
- **高中级** - AI 进阶阶段 (12-18 个月)
- **大学级** - AI 专业阶段 (18-24 个月)

## 🚀 部署到 Vercel

### 方法 1: Vercel CLI (推荐)

```bash
# 安装 Vercel CLI
npm install -g vercel

# 登录 Vercel
vercel login

# 部署
cd output/ai_learning_site
vercel --prod
```

### 方法 2: GitHub + Vercel

1. 创建 GitHub 仓库
2. 推送代码
3. 在 Vercel 导入仓库
4. 自动部署

### 方法 3: Vercel Dashboard

1. 访问 https://vercel.com/new
2. 导入项目
3. 选择静态网站
4. 部署
